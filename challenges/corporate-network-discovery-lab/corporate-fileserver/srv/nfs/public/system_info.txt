Corporate Network Infrastructure
===============================

Server: corporate-fileserver
Role: Multi-service file server
Environment: Production/Testing hybrid

Active Services:
- File Transfer Protocol (FTP)
- Secure Shell (SSH) 
- Server Message Block (SMB)
- Network File System (NFS)
- Simple Network Management Protocol (SNMP)
- HyperText Transfer Protocol (HTTP) on port 26

Security Notes:
- Regular security audits required
- Test configurations may have relaxed security
- Monitor for unauthorized access attempts
- Backup files contain sensitive information
