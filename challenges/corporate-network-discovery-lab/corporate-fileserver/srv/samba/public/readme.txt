Corporate File Server - Public Share
=====================================

This share contains publicly accessible files for the corporate network.

Services running on this server:
- FTP (port 21) - File transfer
- SSH (port 22) - Secure shell
- SMB (ports 139, 445) - File sharing
- NFS (port 2049) - Network file system
- SNMP (port 161) - Network monitoring
- HTTP (port 26) - Testing environment

For support, contact: admin@corporate.local

Note: Some services may have development configurations enabled.
