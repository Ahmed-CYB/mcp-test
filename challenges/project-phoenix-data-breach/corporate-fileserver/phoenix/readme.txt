Project Phoenix FTP Repository
=============================

This FTP server contains development files for
Project Phoenix. Please follow security protocols
when accessing sensitive information.

Directory Structure:
/phoenix/          - Main project files
/phoenix/secure/   - Encrypted sensitive data
/phoenix/docs/     - Public documentation
/phoenix/logs/     - System logs (read-only)

Security Notes:
- All sensitive files are encrypted
- Use standard project encryption keys
- Report any security issues immediately
- Access is logged and monitored

For questions contact: devops@techcorp.local

IMPORTANT: This is a testing environment.
Some security measures may be relaxed for
development purposes. Do not use in production.
