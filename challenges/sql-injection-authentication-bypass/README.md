# SQL Injection Authentication Bypass

## Learning Objectives
- Understand SQL Injection
- Exploit a vulnerable login form to bypass authentication

## Scenario
A corporate login portal is at risk. Exploit the login form to bypass authentication using SQL Injection.

## Access
- Web application: http://victim:8080
- Use Kali Linux for attacking.

## Flag
Authenticate as the admin to retrieve the flag: `CTF{sql_injection_success}`
