# Nmap Skill Practice CTF

## Challenge Overview

Welcome to the Nmap Skill Practice Challenge! This environment is designed to help you hone your skills with the Nmap tool by scanning a simulated network environment.

## Objectives

- Perform a basic network scan.
- Identify open ports and services.
- Detect service versions.
- Conduct OS detection.

## Getting Started

1. **Setup Environment**: Use the provided Dockerfile to set up your environment.
2. **Instructions**: Follow the tasks outlined below using Nmap.
3. **Resources**: Utilize Nmap documentation and cheat sheets for support.

## Tasks

1. Run a basic ping scan to find live hosts in the network.
2. Identify open ports on each live host.
3. Use Nmap scripts to detect service versions.
4. Perform OS detection on the identified hosts.

## Flag Submission

Complete the tasks and provide the required outputs to simulate a realistic attack scenario.

## Good Luck!

Explore, learn, and improve your Nmap skills with this challenge!
