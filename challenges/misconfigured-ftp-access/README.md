# Misconfigured FTP Access

## Learning Objectives
- Understand FTP server configurations
- Identify misconfigurations in FTP services
- Practice exploiting FTP misconfigurations to access restricted files

## Scenario
You have access to an FTP service running internally within a company network. Due to a possible misconfiguration, you may be able to access sensitive files without the need for authentication.

## Hints
1. Explore the FTP service capabilities using anonymous access.
2. Check the directory structure carefully to find any accessible files.
3. Remember the default behaviors of FTP servers when handling anonymous users.
