# SQL Injection Shopping Portal Challenge

## Learning Objectives
- Understand the basics of SQL injection
- Learn how to exploit SQL vulnerabilities to bypass authentication

## Challenge Description
A small online shopping portal has been set up with a simple login system. However, the developers failed to implement proper security measures, leaving the application vulnerable. Your task is to exploit an SQL injection vulnerability to gain unauthorized access to the admin panel.

## Accessing the Challenge
- Access the application at: http://victim:8080
- Use the Kali Linux container to perform your attack

## Flag
Find the flag by accessing the admin panel with the SQL injection vulnerability.
