# XSS, FTP, SSH and VSFTPD Vulnerability Challenge

## Learning Objectives
- Understand and exploit Cross-Site Scripting (XSS) vulnerabilities.
- Gain familiarity with FTP misconfigurations and known backdoors in VSFTPD.
- Learn to exploit weak SSH configurations.

## Challenge Description
Participants will navigate a web application vulnerable to XSS, and exploit FTP and SSH services with known vulnerabilities and weak configurations to retrieve the flag.

Access the challenge environment through the provided Kali Linux desktop and connect to the victim services internally.
