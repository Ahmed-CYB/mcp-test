=== PROJECT ARCHIVE FTP SERVER ===

Welcome to the ProjectArchive FTP Service

This server contains:
- Public project documentation
- Legacy system archives
- Development resources

Access Levels:
- Anonymous users: /public directory
- Authenticated users: /private directory

IMPORTANT SECURITY NOTICE:
All sensitive development files are stored in the private section.
System administrators should ensure proper access controls are in place.

For technical support, contact: admin@projectarchive.corp

Last Updated: March 2024
