Corporate FTP Server - File Access Guidelines

Welcome to the corporate file sharing system.
This server contains employee data and system backups.

Directory Structure:
- /public - General access files
- /employees - Employee data (restricted)
- /backups - System backups (admin only)
- /temp - Temporary file uploads

For technical support, contact IT department.

Note: Anonymous access is enabled for public directory only.
Authenticated users have access to their designated directories.
