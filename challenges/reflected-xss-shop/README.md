# Reflected XSS Shop Challenge

## Learning Objectives
- Understand what reflected XSS vulnerabilities are and how they can be exploited
- Learn how to identify and exploit reflected XSS vulnerabilities
- Practice secure coding to prevent XSS attacks

## Access Instructions
- Use the Kali Linux GUI to attack the victim service at http://victim:8080

## Description
This challenge is a simple online store with a search feature vulnerable to reflected XSS. Your goal is to exploit the vulnerability to find the flag stored in the system.

## Hints
1. Look at how user input is handled and displayed.
2. Reflected XSS appears in the search results.
3. Try manipulating the search query to execute JavaScript.
