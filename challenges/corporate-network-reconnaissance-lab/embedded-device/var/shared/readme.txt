NetStore-3000 Shared Repository
===============================

This directory contains shared files accessible via the web interface.

Directory Structure:
- documents/     - Corporate documents and reports
- software/      - Approved software packages
- backups/       - System configuration backups
- temp/          - Temporary file storage

Access Instructions:
1. Navigate to http://device-ip:28/files/ to browse files
2. Use appropriate credentials for restricted areas
3. Contact IT support for access issues

Security Notice:
All file access is logged and monitored. Unauthorized access attempts
will be reported to the security team.
