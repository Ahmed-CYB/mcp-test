Blueprint Heist - Construction Corp FTP Challenge

Objective: Access the encrypted blueprint files and recover the hidden flag

Starting Point:
1. The FTP server allows anonymous access
2. Look for files that might contain encryption clues
3. The construction company uses standard encryption practices

Hints:
- Company information might reveal encryption details
- System logs often contain useful information
- XOR encryption requires a key of specific length
- Hex encoding is commonly used for encrypted data
- The key might be hidden in plain sight

Tools you might need:
- FTP client
- Hex decoder
- XOR decryption tools
- Text analysis utilities
