# The Encrypted Enigma

Welcome Crypto Analyst!

Your mission is to decrypt a series of encrypted messages left behind by the mysterious organization known as The Cipher Crew. These messages contain information critical to uncovering the location of the elusive secret flag.

## Objective

Utilize your cryptographic skills to crack the encoded messages and find the secret flag.

### Rules:
1. Share discoveries, not solutions.
2. Collaborate and discuss techniques, but the path to the flag is yours to tread.
 
### Setup Instructions

1. Clone this repository to access the challenge files.
2. Analyze the provided encoded files in the `messages` directory.
3. Use any cryptographic tools or techniques at your disposal to decipher the messages.

### Hints
- Each message might require a different cryptographic approach.
- Pay attention to file names and comments for potential clues.

Good luck unraveling the enigma!
