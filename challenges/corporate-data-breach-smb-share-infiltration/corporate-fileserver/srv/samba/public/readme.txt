Corporate File Server - Public Access
=====================================

Welcome to the Corporate File Server

Available Shares:
- public: General company documents
- hr-docs: Human Resources (restricted)
- finance: Financial documents (authorized users only)
- backup: System backups (maintenance)
- legacy-system: Archived legacy files

For technical support, contact IT helpdesk.

Security Notice: Unauthorized access is monitored and logged.
All activities are subject to corporate security policies.
