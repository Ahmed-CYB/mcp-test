MEMORY FORENSICS ANALYSIS - VOLATILITY 2 GUIDE

STEP 1: Identify memory profile
vol.py -f memory.raw imageinfo

STEP 2: List running processes
vol.py -f memory.raw --profile=Win7SP1x64 pslist

STEP 3: Examine process tree
vol.py -f memory.raw --profile=Win7SP1x64 pstree

STEP 4: Find suspicious process (PID 2048)
vol.py -f memory.raw --profile=Win7SP1x64 pslist | grep backdoor

STEP 5: Extract command line arguments
vol.py -f memory.raw --profile=Win7SP1x64 cmdline

STEP 6: Look for the flag in process memory
vol.py -f memory.raw --profile=Win7SP1x64 memdump -p 2156 --dump-dir ./

STEP 7: Search extracted memory for flag
strings 2156.dmp | grep CTF

STEP 8: Alternative - search all memory
strings memory.raw | grep -i ctf
