# FTP Weak Credentials Challenge

## Description
This challenge involves exploiting a vulnerable FTP server that has been misconfigured with weak credentials. Participants need to identify the credentials used to access the server and retrieve the flag stored within.

## Learning Objectives
- Understand weak credential vulnerabilities.
- Practice exploiting FTP servers.
- Learn about common FTP misconfigurations.

## Setup
- Deploy the challenge using the provided `docker-compose.yml`.
- Access the FTP server using any FTP client.
- Use the Kali Linux container to perform the attack.

## Hint
- Consider default or common passwords for FTP access.

## Access
- Kali Linux is available through a web browser with VNC access. The password is `password`.
- The FTP server is accessible at `ftp://172.20.0.10`.
