# Corporate Network Challenge

You've discovered an internal file server through SNMP enumeration of the corporate gateway. The server contains encrypted database backups that may hold sensitive information.

## Discovered Information:
- Internal file server: 192.168.1.100
- SMB shares are accessible
- Database backup is encrypted using classical methods
- Server configuration hints at the encryption parameters

## Your Task:
Decrypt the database backup to retrieve the flag.

## Hints:
- The encryption is a classical transposition cipher
- Pay attention to the server room number mentioned in the config
- The data appears to be arranged in a specific pattern
