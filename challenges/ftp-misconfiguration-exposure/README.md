# FTP Misconfiguration Exposure

## Challenge Description
An FTP server has been set up, but due to misconfigurations in the vsftpd settings, sensitive files are exposed to unauthorized access. Your mission is to explore the FTP service and find the flag.

## Learning Objectives
- Understand FTP server configurations
- Learn about common FTP server misconfigurations and their implications
- Practice accessing misconfigured FTP services from a networked environment

## Instructions
- Access the attacker environment through the Kali Linux GUI provided.
- Investigate the FTP service running on ftpserver (accessible at 172.21.0.10).
- Find and retrieve the flag stored in the FTP server.
