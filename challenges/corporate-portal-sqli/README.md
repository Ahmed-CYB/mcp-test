# Corporate Portal SQLi Challenge

## Learning Objectives
- Understand the implications of SQL Injection vulnerabilities
- Learn to exploit SQL Injection to bypass authentication
- Practice analyzing and modifying SQL queries dynamically

## Instructions
- You are tasked to test the security of the XYZ Corp's login portal.
- Analyze the login mechanism and identify potential vulnerabilities.
- Your goal is to bypass the login and access the portal without valid credentials.

Access the application via `http://victim:8080` from the Kali container.

## Hints
1. Pay attention to how user input is being handled in SQL queries.
2. Experiment with SQL Injection techniques to bypass authentication.
3. Consider what happens when the query returns more than one result.
