# FTP Hidden Treasure

## Learning Objectives
- Understand how FTP permissions can be misconfigured
- Learn to navigate and search directories on an FTP server

## Scenario
A file-sharing company has an FTP server with loose permissions. Your mission is to explore the FTP directory structure and find a hidden treasure file containing the flag.

## Access Instructions
- Launch the Kali Linux container to start attacking.
- Access the FTP server at `ftp://172.21.0.10` using the credentials below:
  - Username: ftpuser
  - Password: password123

## Hints
1. Start by logging in with the provided credentials.
2. Explore all directories and files.
3. Look for a file that seems out of place or hidden.

## Notes
The flag is located in a file that is not immediately visible and requires some directory exploration.
