ACME CORPORATION FTP SERVER
===========================

Welcome to the ACME Corporation file server.

This server contains corporate documents and resources.
Access is provided for internal use only.

DIRECTORY STRUCTURE:
/corporate/
  ├── policies/     - Corporate policies and procedures
  ├── hr/          - Human resources documents
  ├── finance/     - Financial reports and data
  ├── it/          - IT documentation and diagrams
  └── executive/   - Executive communications

/systems/
  ├── backups/     - Backup configurations
  ├── inventory/   - System inventory files
  └── logs/        - System log files

/public/
  └── marketing/   - Public marketing materials

For technical support, contact: it-support@acme.corp
For access issues, contact: admin@acme.corp

Last updated: February 22, 2024

NOTE: This server allows anonymous access for internal convenience.
All activities are logged and monitored.
