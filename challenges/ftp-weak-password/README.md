# FTP Weak Password Challenge

## Description
A small company has set up an FTP server for file sharing, but they've made a critical mistake in their password policy. Your task is to exploit the weak password setup to gain access and retrieve the hidden flag.

## Learning Objectives
- Understand the implications of weak password policies
- Learn how to use common FTP tools to exploit weak passwords

## Access Instructions
1. Use the Kali Linux environment provided to attack the FTP server running at `ftpserver` (172.25.0.10).
2. The user credentials are weak and can be easily cracked.
3. Retrieve the flag from the FTP server.

## Flag
`CTF{weak_password_exploited}`
