# FTP Hidden Flag Extraction

## Challenge Description

Your task is to exploit a misconfigured FTP server to extract a hidden flag. The server allows anonymous access, and you must explore the possibility of hidden or poorly secured files.

## Learning Objectives
- Understand FTP server configuration and common misconfigurations.
- Learn to explore FTP servers for accessible data.
- Practice using FTP clients to interact with servers.

## Hints
1. Try connecting anonymously to see what is exposed.
2. Explore hidden directories or files that might not be listed at first glance.
3. Look for classic mistakes in FTP configurations that might expose sensitive files.

## Accessing the Challenge
- Attack from Kali Linux (access the FTP server at ftp://ftpserver or 172.25.0.10).
