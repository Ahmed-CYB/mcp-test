# SQL Injection Corporate Portal

## Learning Objectives
- Understand SQL Injection vulnerabilities
- Exploit SQL Injection to bypass authentication
- Use SQL Injection techniques in a realistic scenario

## Challenge Overview
You are provided with access to a corporate login portal. Use your skills to exploit SQL Injection vulnerabilities in the login form to gain unauthorized access.

## Instructions
- Access the challenge at http://victim:8080
- Use the Kali Linux container to perform the attack

## Tips
- Pay attention to error messages
- Consider different SQL Injection techniques
