CorpNet Corporate Web Server

Welcome to the CorpNet corporate network infrastructure.

Available Services:

  • HTTP Web Server (Port 80)
  • HTTPS Secure Web (Port 443)
  • SSH Remote Access (Port 22)
  • FTP File Transfer (Port 21)
  • MySQL Database (Port 3306)
  • SNMP Monitoring (Port 161)
  • Legacy Telnet Management (Non-standard port)
Security Notice: Legacy telnet service maintained for backward compatibility with older management tools. Modern SSH recommended for new connections.

System Information:

  • Server: linux-web-server.corpnet.local
  • OS: Ubuntu Server 20.04 LTS
  • Last Updated: March 2019
  • Status: Production
