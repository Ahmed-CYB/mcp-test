# Nmap Skill Practice CTF

## Introduction

Welcome to the Nmap Practice Challenge! This CTF is designed to help you hone your skills with the Nmap tool. You'll explore a simulated network environment, identify open ports, detect services, and map the network layout.

## Objectives

- Utilize Nmap to perform a basic scan of the network.
- Identify all open ports on the target machine.
- Detect versions of services running on open ports.
- Perform OS detection and subnet mapping.

## Instructions

1. Start the Docker container using the provided Dockerfile.
2. Use Nmap to scan the IP address of the running container.
3. Document each step of your process and findings.
4. Submit your findings along with any cool tips or tricks you learned during the process.

## Tools Required

- Nmap (Network Mapper)

## Hints

- Remember to explore different Nmap flags and options.
- Use verbose mode for more detailed output.
