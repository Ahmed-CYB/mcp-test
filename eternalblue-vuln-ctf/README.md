# EternalBlue Vulnerability CTF

## Overview
This Capture The Flag (CTF) challenge is designed to simulate the exploitation of the EternalBlue vulnerability.

## Instructions
1. Build the Docker image using the provided Dockerfile.
2. Run the Docker container.
3. Attempt to exploit the exposed SMB service.
4. Capture the flag hidden in the system.

## Note
This setup is for educational purposes only. Please ensure to run this in a controlled, isolated environment.
